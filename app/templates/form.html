Form Demo — POST Form Data

Route: GET /form
Concept: HTML forms collect user input. When the form uses method="post", the browser sends field values in the request body — not in the URL.
Before submitting: When you click Submit, your browser will send a POST request to /form with the field values in the request body. FastAPI uses python-multipart to parse those fields.
