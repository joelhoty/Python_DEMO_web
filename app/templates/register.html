What will happen when you submit:
  1. FastAPI reads username and password from the POST body.
  2. A random 16-byte salt is generated with secrets.token_hex(16).
  3. The password is hashed: hashlib.pbkdf2_hmac("sha256", password, salt, 260000).
  4. Only salt$hash_hex is stored in the database — never the password.
