Database Demo — SQLite with Python

Route: GET /database
Concept: This site stores users in a SQLite database. SQLite is a file-based relational database that requires no separate server. Python's standard-library sqlite3 module is used — no ORM needed.

How the users table is created

CREATE TABLE IF NOT EXISTS users (
    id           INTEGER PRIMARY KEY AUTOINCREMENT,
    username     TEXT    NOT NULL UNIQUE,
    password_hash TEXT   NOT NULL,
    created_at   TEXT    NOT NULL
);

Try it: look up a user by ID

Register a user first

Use the Register page to create a user, then come back here to look them up by ID.

What Python does:
  1. Open a connection to teaching.db (the SQLite file).
  2. Execute a parameterized SQL query with ? placeholders — never string formatting (SQL injection risk).
  3. Return the row as a sqlite3.Row object.
  4. Close the connection.
